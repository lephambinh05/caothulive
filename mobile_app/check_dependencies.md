## üöÄ Build Commands for macOS

### 1. Setup (First time)
```bash
cd mobile_app
chmod +x ios_build_check.sh
./ios_build_check.sh
```

### 2. Manual Setup
```bash
cd mobile_app
flutter clean
flutter pub get
cd ios
rm -rf Pods Podfile.lock .symlinks
pod repo update
pod install --repo-update --verbose
chmod -R 755 Pods/ Flutter/
cd ..
flutter build ios --no-codesign
```

### 3. Run on Simulator
```bash
flutter run -d ios
```
